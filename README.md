Motivational app 
================================

Introduction
------------
The Motivation app contains a scrollable list of 10 cards.

Getting Started
---------------
1. Install Android Studio, if you don't already have it.
2. Download the project.
3. Import the project into Android Studio.
4. Build and run the project.
